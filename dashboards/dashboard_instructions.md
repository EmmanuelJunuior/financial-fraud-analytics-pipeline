### Fraud Analytics Dashboard Instructions

This dashboard displays key metrics and alerts for financial fraud transactions. It uses the outputs generated by the fraud_transactions_pipeline Airflow DAG.

# 1. Prerequisites

Python 3.10+

Pandas

Plotly
 or Dash
 for interactive dashboards

The DAG should have run successfully at least once: fraud_transactions_pipeline

fraud_alerts.csv should exist in data/ folder

# 2. Folder Structure

financial-fraud-analytics-pipeline/
├── airflow/
│   └── dags/

├── dashboard/

# 3. Running the Dashboard
Option 1: Using Python Script

1. Navigate to the dashboard folder:
```
cd dashboard
```
2. Run the dashboard script:

python dashboard.py

The dashboard will start a local server, usually at:
http://127.0.0.1:8050

You can view fraud metrics such as:

Number of fraudulent transactions

Total amount flagged

Transaction types with highest fraud count

Recent alerts

Option 2: Using Jupyter Notebook / Colab

Load the alerts CSV:

import pandas as pd

alerts = pd.read_csv('../data/fraud_alerts.csv')
alerts.head()

Visualize with Plotly:

import plotly.express as px

fig = px.bar(alerts, x='type', y='total_amount', color='fraud_count',
             title='Fraud Transactions Alerts')
fig.show()
4. Updating the Dashboard

Ensure the Airflow DAG runs daily to refresh fraud_alerts.csv.

Re-run the dashboard to see updated metrics.

If using a database backend instead of CSV, update the connection in dashboard.py.

5. Key Notes

The dashboard reads outputs from your ETL & dbt pipeline.

Only transactions flagged in fct_fraud_summary as potential fraud are visualized.

Alerts are generated using thresholds (e.g., amount > 200,000 or fraud_count > 0) in the DAG.
